# simulador-de-ponto
é um simulador de ponto e salário com clima, cargos e regras, com:  ✔ Sistema de login/registro via JSON ✔ Dias de trabalho por cargo ✔ Simulação diária com clima ✔ Justificativa de falta com palavras-chave ✔ Ciclo financeiro de 30 dias ✔ Interface Tkinter moderna ✔ Persistência de dados ✔ Cargos pré-definidos
